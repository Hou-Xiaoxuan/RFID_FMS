# 标签命名

EPC前8位(16进制范围0~4294967295)作为唯一识别ID，之后的位数作为可扩展的控制位。

第九位：识别标签种类

| value | meaning    |
| ----- | ---------- |
| 0     | 方形标签   |
| 1     | 矩形标签   |
| 2     | 细条形标签 |
